# 微博浮图控件 - miniblogImgPop

针对微博网站,鼠标移过小图弹出浮动大图的脚本,让您看微博图片时鼠标不再滴滴答答。

支持新浪微博，腾讯微博，搜狐微博，网易微博，人民微博，体坛微博，百度说吧,天涯微博,QQ空间,嘀咕网,点点网,新浪轻博等微博网站。 

按住Z键可看超屏大图。
